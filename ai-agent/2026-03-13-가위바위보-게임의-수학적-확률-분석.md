### Context
가위바위보(Rock, Paper, Scissors) 게임은 대표적인 제로섬 게임(Zero-sum game)으로, 각 플레이어는 세 가지 선택지(가위, 바위, 보)를 가집니다. 특정 전략 없이 무작위로 선택할 때 승리 확률이 어떻게 계산되는지, 그리고 게임 이론적 관점에서 어떤 의미를 갖는지 정리합니다.

### Core
각 플레이어가 선택할 수 있는 경우의 수는 총 9가지이며, 이를 행렬로 표현하면 다음과 같습니다.

```
플레이어 1 \ 플레이어 2 | 가위 | 바위 | 보
-------------------------------------------
가위                  | 비김 | 패배 | 승리
바위                  | 승리 | 비김 | 패배
보                    | 패배 | 승리 | 비김
```

각 선택지별 결과 분포는 다음과 같습니다.
* 승리(Win): 3가지 경우 (가위vs보, 바위vs가위, 보vs바위)
* 패배(Loss): 3가지 경우 (가위vs바위, 바위vs보, 보vs가위)
* 비김(Draw): 3가지 경우 (가위vs가위, 바위vs바위, 보vs보)

확률 계산식:
* 승리 확률 = 승리 경우의 수(3) / 전체 경우의 수(9) = 1/3 (약 33.3%)
* 비기지 않을 확률 = (승리 + 패배) / 전체 = 6/9 = 2/3 (약 66.7%)

### Insight
가위바위보는 각 선택지의 확률이 동일하다는 가정하에 완벽한 균형(Nash Equilibrium)을 이루고 있습니다. 수학적으로 어느 한쪽이 우위를 점할 수 없는 구조이며, 따라서 정적인 상태에서의 승리 확률은 1/3에 수렴합니다. 다만, 실제 인간 간의 대결에서는 심리적 요인이나 패턴 인식에 의해 선택의 무작위성이 깨질 수 있으므로, 게임 이론(Game Theory)적 관점보다 심리적 전략이 개입되는 비정형 데이터 분석의 영역으로 확장될 수 있습니다.
