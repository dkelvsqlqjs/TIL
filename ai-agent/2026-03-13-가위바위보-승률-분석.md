### Context
가위바위보(Rock Paper Scissors)는 3가지 선택지가 있는 제로섬 게임(Zero-sum game)입니다. 각 플레이어가 무작위로 선택을 한다고 가정할 때, 개인이 특정 라운드에서 승리, 패배, 혹은 비길 확률을 수학적으로 계산하고자 합니다.

### Core
확률적 관점에서 각 플레이어가 가위(Scissors), 바위(Rock), 보(Paper)를 선택할 확률은 각각 1/3로 동일합니다. 총 경우의 수는 3 × 3 = 9가지입니다.

* 비기는 경우 (Tie): (가위, 가위), (바위, 바위), (보, 보) -> 3가지
* 플레이어 A가 이기는 경우 (Win): (바위, 가위), (가위, 보), (보, 바위) -> 3가지
* 플레이어 A가 지는 경우 (Loss): (가위, 바위), (보, 가위), (바위, 보) -> 3가지

각 상황별 확률:
* P(Win) = 3/9 = 1/3 (약 33.3%)
* P(Loss) = 3/9 = 1/3 (약 33.3%)
* P(Tie) = 3/9 = 1/3 (약 33.3%)

### Insight
완벽하게 무작위(Random)로 선택한다고 가정할 때, 가위바위보의 승률은 정확히 1/3입니다. 하지만 실제 인간은 심리적인 요인(Psychological factors)과 패턴(Patterns)을 보이기 때문에 순수 확률과는 차이가 발생합니다. 예를 들어, 패배한 플레이어는 다음 라운드에서 이전에 낸 패를 바꾸려는 경향이 있고, 승리한 플레이어는 이전의 승리한 패를 반복하려는 경향이 있습니다. 이러한 심리적 편향(Biases)을 이용하는 에이전트를 구축한다면 1/3 이상의 승률을 확보할 수 있습니다.
